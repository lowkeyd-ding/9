# 九宫格数独 - 儿童练习版

一个专为二年级孩子设计的数独练习应用，界面友好，功能完善，让孩子在游戏中锻炼逻辑思维能力。

## 🎮 在线体验

访问 GitHub Pages: [点击这里体验](https://你的用户名.github.io/sudoku-kids/)

## ✨ 功能特点

- **🎯 三种难度等级**
  - 简单 ⭐ - 适合初学者
  - 中等 ⭐⭐ - 适合有一定基础的孩子
  - 困难 ⭐⭐⭐ - 适合挑战

- **💡 智能辅助**
  - 提示功能 - 随机给出正确答案
  - 检查功能 - 实时检查答案正确性
  - 错误提醒 - 自动标记错误答案

- **⏱️ 游戏统计**
  - 实时计时器
  - 错误计数器
  - 完成时间记录

- **🎉 趣味互动**
  - 完成时的五彩纸屑庆祝动画
  - 鼓励性的文字提示
  - 色彩鲜艳的儿童友好界面

- **⌨️ 多种输入方式**
  - 点击屏幕按钮输入
  - 键盘数字键输入（1-9）
  - 支持擦除功能

- **📱 响应式设计**
  - 完美适配手机、平板、电脑
  - 流畅的动画效果

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/你的用户名/sudoku-kids.git
```

2. 打开 `index.html` 文件即可开始游戏

或者直接访问 GitHub Pages 在线体验！

## 📂 项目结构

```
sudoku-kids/
├── index.html    # 主页面
├── style.css     # 样式文件
├── script.js     # 游戏逻辑
└── README.md     # 说明文档
```

## 🎓 适用人群

- 小学二年级及以上学生
- 想要锻炼逻辑思维的孩子
- 数独爱好者

## 🛠️ 技术栈

- HTML5
- CSS3
- 原生 JavaScript（无依赖）

## 📝 游戏规则

1. 每行都包含1-9的数字，不重复
2. 每列都包含1-9的数字，不重复
3. 每个3×3的宫格内包含1-9的数字，不重复
